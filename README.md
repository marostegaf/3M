#### Funcionalidades
`Adicionar Produto` Você pode inserir um novo produto. <br/>
`Excluir` Você pode excluir um produto.
```bash
{
  "nome": "Moletom Branco",
  "price": 89,
  "image": "http://link-da-imagem.com/imagem.jpg"
}
```
###### Você pode `Adicionar` ou `Remover` um produto que todos poderam ver!
